# 🌊 Batalha Naval

Projeto desenvolvido em C com o objetivo de simular um tabuleiro de Batalha Naval com posicionamento de navios e habilidades especiais.

# 📌 Descrição
O projeto foi desenvolvido em 3 níveis progressivos, com foco no raciocínio lógico e no uso de estruturas fundamentais da linguagem C.
- Matrizes
- Estruturas de repetição
- Loops aninhados
- Condicionais
- Manipulação de coordenadas
- Validação de limites do tabuleiro

# 🛠️ Funcionalidades
🌊 Representação do tabuleiro (10x10)
🚢 Posicionamento de navios (valor 3)
✨ Áreas de efeito das habilidades (valor 5)
💧 Água representada por (valor 0)
📍 Sistema de coordenadas com letras (A-J) e números (1-10)
🧠 Sobreposição de habilidades sem sobrescrever navios
✅ Validação de limites do tabuleiro
